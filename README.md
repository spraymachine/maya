# Dr. Sridhar Akiri - Academic Web Profile

A clean and minimalistic web profile built with HTML, CSS, and JavaScript, featuring a green and light color palette.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, minimalistic design with smooth animations
- **Green Color Palette**: Professional green and light color scheme
- **Interactive Elements**: Smooth scrolling, hover effects, and animations
- **Professional Sections**:
  - Hero section with profile photo
  - About section with statistics
  - Education timeline
  - Professional experience
  - Publications and research areas
  - Contact information

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript**: Interactive features and smooth animations
- **Font Awesome**: Icons for visual enhancement
- **Google Fonts**: Inter font family for typography

## File Structure

```
maya/

├── script.js          # JavaScript for interactivity
```

- Chrome (latest)
- Firefox (latest)
- Safari (latest)

## Usage

Simply open `index.html` in any modern web browser to view the profile.

## Customization

The website uses CSS custom properties (variables) for easy customization of colors, spacing, and typography. Modify the `:root` section in `style.css` to change the design.
